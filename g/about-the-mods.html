Server Mods Overview

Orgins Mod

The Origins Mod is a mod that allows players to choose from a variety of different origins. Each origin has its own unique abilities and playstyle, providing a fresh and exciting experience for players. View the wiki for more information on the different origins and their abilities. Choose an Origin and begin your game with special abilities, some of which help you out, while others hinder you in your gameplay. The different Origins all have a different impact on the vanilla gameplay. If you don't like these at all, you can also pick the "Human" origin while playing with friends on a server.

Friends&Foes

Friends&Foes adds outvoted and forgotten mobs from the mob vote. One of the main goals is to further expand upon original concepts and add new vanilla-like features related to all the mobs. You can check wiki for a detailed overview of implemented mobs and features.

YUNG's Better Features

YUNG's Better Features is a mod that changes pre-existing features in Minecraft and updating them to be more modern and sophisticated. This mod is a collection of features that are designed to improve the game's quality of life and add new features that are not present in the base game. The mod is server-side only and not required to be installed on the player's client.

Inky Addons (INKA)

A custom coded mod for the Inky SMP server. This mod adds custom items and features to the server. It is required to play on the server. Additionally this mod contains MANTIS, which is an adminstration tool for the developers of the server that helps link the minecraft server and all of its players to outside databases and applications. By playing on the server you agree to the logging of your data for the purpose of server administration. Learn More Here.
