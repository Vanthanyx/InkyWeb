How to Install Fabric for Minecraft

  1. Download the Fabric Installer:

    Go to the Official Fabric Website and download the Fabric Installer for your operating system.

  2. Run the Fabric Installer:

    Open the downloaded Fabric installer file. You may need to allow the file to run if prompted by your operating system.

  3. Select Minecraft Version:

    In the Fabric installer, select the Minecraft version you want to install Fabric for. Make sure to choose the correct version that matches the server. Currently: Minecraft 1.21.1 | Fabric 0.16.9. Remember your Launcher Location, it should look like this: C:\Users\USERNAME\AppData\Roaming\.minecraft

  4. Install Fabric:

    Click the "Install" button in the Fabric installer. The installer will download and set up the necessary files for Fabric.

  5. Open Minecraft Launcher:

    Open the Minecraft Launcher. You should see a new profile named "fabric-loader" with the version you selected.

  6. Install The Server Mods:

    The server mods can be found in the Info Channel of the discord server. Download the mods and place them in the mods folder. Typically located at: C:\Users\USERNAME\AppData\Roaming\.minecraft\mods. You may need to make the folder yourself.

  7. Launch Minecraft with Fabric:

    Select the "fabric-loader" profile and click "Play" to start Minecraft with Fabric installed.
